# PROVA PRÁTICA PARA PROGRAMADOR PHP II


### Configure a URL da API em /src/configs/api.js
```
URL_API = http://localhost
```

### Instalar NPM
```
npm install
```

### Compilar para Desenvolvimento
```
npm run serve
```

### Compilar para Produção
```
npm run build
```

Acesse o projeto
[http://localhost:8082](http://localhost:8082)
